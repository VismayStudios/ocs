    Global

    Auth(config)

    Main Auth class for Authentication

    new Auth(config)

    create a new Auth Object sets config for the instance sets authSessionId for the instance
    Parameters:
    Name Type Description
    config Object Instance of Config class.

    Methods

    authenticate() → {function}

    Authenticate middleware to be used within routes
    Returns:
    function - Middleware which accepts binding of request, response and next callbacks

    async createAuthSession() → {String}

    create a new authSession for a user
    Parameters:
    Name Type Description
    authSessionData.userId ObjectId User Id
    authSessionData.deviceId String Unique device Id
    authSessionData.deviceToken String Device Token
    authSessionData.deviceType String Device type [IOS, ANDROID, WEB]
    authSessionData.ip String Ip address of request
    Returns:
    String - accessToken

    generateToken(authSessionId) → {String}

    generates an access token for a authSessionId id
    Parameters:
    Name Type Description
    authSessionId String authSessionId from mongo authSession collection
    Returns:
    String - accessToken

    logout(flushAll) → {function}

    Destroy authSession(s) request must be authenticated before calling logout
    Parameters:
    Name Type Default Description
    flushAll boolean false logout user from all authSessions
    Returns:
    function - express middleware/handler

    async validate() → {Object}

    validate a authSessionId Id
    Throws:
    AuthSession does not exist {if no authSession found}
    Type
    Error
    Returns:
    Object - valid authSession Object
